# Minblcode Homepage

`minblcode.com`에 배포되는 Minblcode 공식 홈페이지 저장소입니다.

## Overview

- 정적 한 페이지 사이트입니다.
- 현재 스택은 `index.html`, `styles.css`, `script.js`입니다.
- GitHub Pages `main` 브랜치 루트(`/`)에서 배포합니다.
- 커스텀 도메인은 `minblcode.com`입니다.

## Brand Context

- Minblcode는 작고 똑똑한 인디 스튜디오 톤을 유지합니다.
- 한글 표기는 `민블코드`를 사용합니다.
- 페이지 카피는 짧고 정돈된 느낌을 우선합니다.
- 브랜드 이름 표기는 항상 `minblcode`입니다.
- 연락 이메일은 현재 `minblecode@gmail.com`을 사용합니다.

## Current Page Structure

1. Hero
2. 소개
3. 프로젝트
4. 문의

현재 공개 내용:

- 출시 프로젝트: `Poku`, `찰떡 제시어 퀴즈`
- 개발 중: `전자액자 앱`

## Content Rules

- 장황한 설명보다 짧고 직접적인 문구를 사용합니다.
- 상단 메뉴는 작고 가볍게 보이도록 유지합니다.
- 브랜드 이름을 화면에 과하게 반복하지 않습니다.
- `소개`는 `프로젝트`보다 먼저 나옵니다.
- 연락 섹션은 이메일만 노출합니다.
- 사이트 본문에는 미래 사업 로드맵이나 GitHub Pages 관련 설명을 넣지 않습니다.
- 과한 스크롤 리빌이나 느린 모션은 사용하지 않습니다.

## Quick Edit Notes

- 로고를 바꿀 때는 본문 이미지와 `og:image`를 함께 확인합니다.
- 새로고침 시 페이지는 항상 맨 위에서 시작하도록 스크립트가 설정되어 있습니다.
- 접근성용 skip link와 섹션 앵커 네비게이션은 유지합니다.
